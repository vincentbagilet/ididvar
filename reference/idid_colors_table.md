# A table with the main colors used in base plots in this package

A table containing the name of the color combination, a base color, a
background color, a lighter and a darker one

## Usage

``` r
idid_colors_table
```

## Format

A data frame with 644 rows and 3 variables:

- pal:

  palette name

- base:

  base color for one color graphs and color for large values in multiple
  color graphs

- mid:

  Middle color in multiple color graphs

- complementary:

  complementary color for one color graphs and color for small values in
  multiple color graphs

- text:

  Text color
